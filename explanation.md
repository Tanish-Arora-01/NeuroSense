# NeuroSense Project Overview

NeuroSense is a comprehensive, AI-powered dementia screening platform that merges classical cognitive test metadata with advanced acoustic speech feature extraction to compute probabilistic risk scores. Architected as a highly modular 3-tier microservice application, it features a React 19 frontend for dynamic clinical dashboards, a Node.js API gateway managing secure OAuth authentication and automated PDF report generation, and a Python FastAPI inference engine powered by privacy-centric, offline Vosk speech-to-text and scikit-learn predictive models. The underlying machine learning model, trained on 1,600 clinical samples, achieves an 81.5% accuracy and an 89.5% ROC-AUC score, demonstrating robust performance in identifying dementia risk. From a software engineering (SDE) perspective, the architecture is engineered for high performance and observability: the offline Vosk model operates with a rapid 0.15s transcription latency, the API gateway ensures fault tolerance through strict proxy timeout controls and exhaustive latency tracing via a dedicated ScreeningAuditLog, and the Vite-powered React SPA maintains a lean client footprint with critical dashboard components optimized to ~31KB for a fast time-to-interactive. Deployed via a cost-effective $0 cloud strategy leveraging PaaS solutions like Vercel and Render, the platform operates as a resilient, distributed ecosystem ensuring robust separation of concerns, secure inter-service proxying, and scalable delivery of machine learning insights to aid healthcare professionals in early diagnostics.
